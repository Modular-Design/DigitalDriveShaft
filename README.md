# DigitalDriveShaft

- Python Version: 3.8+
- Future: 3.10+
- 3.10 will allow slicker code, but is not needed yet

## Installation

0. setup venv `python -m venv .`
1. activate venv (Windows: `venv\Scripts\activate`, Linux)
2. install requrements with `pip install -e .`


## Testing

run `pytest` in venv
